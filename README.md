# Message for Humanity

A high-end, responsive landing page for Message for Humanity - a non-profit organization focused on sharing guidance and peace.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (styling)
- **Framer Motion** (scroll animations and reveals)
- **Lucide React** (icons)
- **TypeScript**

## Features

- Modern, premium design inspired by Apple, Calm, and Pfida
- Smooth scroll animations and reveals
- Responsive design (mobile-first)
- Video background hero section
- Audio playback functionality
- Expandable solution cards
- Dark blue accent color scheme

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/page.tsx` - Main landing page
- `src/app/globals.css` - Global styles and CSS variables
- `src/app/layout.tsx` - Root layout with fonts
- `public/` - Static assets (images, video, audio)

## Color Palette

- Background: `#f9f8f3` (Warm Paper/Sand)
- Text: `#1c1c1e` (Soft Charcoal)
- Accent: `#40516a` (Dark Blue)
- Accent Dark: `#242d3d`
- Accent Darker: `#0b132b`
